# Biblioteca de Prompts — WGEasy

Esta pasta contem prompts especializados para diferentes contextos de trabalho.

## Prompts Disponiveis

| Arquivo | Especialista | Ativacao |
|---------|-------------|----------|
| `ESPECIALISTA-FINANCEIRO-OBRAS.md` | Auditor Financeiro de Obras | `especialista financeiro` |
| `ESPECIALISTA-FINANCEIRO-PESSOAL.md` | Consultor Financeiro Pessoal | `avaliacao trimestral` |

## Como Usar

Quando precisar de um especialista específico, mencione o comando de ativação correspondente.

### Exemplos:
```
"especialista financeiro, analise a pasta do cliente Marina Wikbold"
"avaliacao trimestral, analise Q4 2024 do William"
```

## Adicionar Novos Prompts

1. Criar arquivo `.md` nesta pasta
2. Seguir o formato padrão com seções:
   - PAPEL DA IA
   - CONTEXTO OPERACIONAL
   - OBJETIVO PRINCIPAL
   - REGRAS DE ANÁLISE
   - FORMATO DE RESPOSTA
   - CRITÉRIO DE DECISÃO
   - COMO ATIVAR
3. Atualizar este README com o novo prompt
